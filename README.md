# Enumerable Ruby methods

> This project is the recreation of some of the methods that are part of the Enumerable module. The project method specifications were created using RSpec.

## Methods recreated:

- .each
- .each_with_index
- .select
- .all?
- .any?
- .none?
- .count
- .map
- .inject

## Built With

- Ruby
- RSpec
- Rubocop
